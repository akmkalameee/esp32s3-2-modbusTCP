# esp32s3-2-modbusTCP
This is ESP32-S3 based Modbus RTU to Modbus TCP (Wifi) Converter. Code written on Arduino IDE. Static IP, Baud Rate, Data Bit, Stop Bit, Parity can be configured from directly Arduino IDE.
